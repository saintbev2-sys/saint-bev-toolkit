Debug Proxy

Intercepts and logs raw HTTP traffic on a local socket. Browsers can't open TCP listeners for security reasons, so this one runs from the terminal, not this page — start it locally with:

python3 saint_bev_toolkit.py

Then select 3 from the menu. It listens on 127.0.0.1:8888 and writes every request/response to proxy_trace.log.
